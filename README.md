# React + TypeScript + Vite

This project is a minimal React + TypeScript setup using [Vite](https://vitejs.dev/) for fast development with HMR (Hot Module Replacement) and ESLint for linting.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v16+ recommended)
- npm (comes with Node) or [Yarn](https://yarnpkg.com/)

### 📦 Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/jasrulete/portfolio.git
cd portfolio
npm install
npm run dev
